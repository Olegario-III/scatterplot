# D3 Scatterplot Visualization

A responsive scatterplot built with **D3.js v7** visualizing doping allegations in professional bicycle racing.

## Project Overview

This project displays the relationship between cyclists' finishing times and their years in the Tour de France, highlighting riders with doping allegations.

### Features

- Interactive scatterplot using D3.js
- Color-coded dots (red = doping allegations, blue = no allegations)
- Hover tooltips with detailed rider information
- Responsive design
- Legend for data interpretation
- Axes with proper labeling

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- [D3.js v7](https://d3js.org/)
